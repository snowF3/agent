# 배포 가이드 — Render + Streamlit Cloud

## 순서: Render(에이전트) 먼저 → Streamlit Cloud(대시보드) 나중에
(Streamlit에서 Render URL을 넣어야 하므로)

---

## Step 1: Render에 Agent API 배포

### 1-1. Render 가입
1. https://render.com 접속
2. **"Get Started for Free"** 클릭
3. **"GitHub"** 로 로그인 (snowF3 계정으로)

### 1-2. New Web Service 생성
1. 로그인 후 대시보드에서 **"New +"** 버튼 클릭
2. **"Web Service"** 선택
3. **"Build and deploy from a Git repository"** 선택 → Next

### 1-3. GitHub 레포 연결
1. **"Connect a repository"** 화면에서
2. GitHub 계정 연결 (처음이면 "Configure account" 클릭)
3. 레포 목록에서 **snowF3/agent** 선택 → **Connect**

### 1-4. 서비스 설정
아래 값들을 정확히 입력:

| 항목 | 값 |
|------|-----|
| **Name** | `xray-agent` |
| **Region** | Singapore (Southeast Asia) 또는 Oregon (US) |
| **Branch** | `prod` ← 중요! |
| **Runtime** | `Python 3` |
| **Build Command** | `pip install -r requirements.txt` |
| **Start Command** | `uvicorn agent.server:app --host 0.0.0.0 --port $PORT` |
| **Instance Type** | **Free** |

### 1-5. 환경변수 설정
스크롤 내려서 **"Environment Variables"** 섹션:

**"Add Environment Variable"** 클릭하고 아래 3개 추가:

| Key | Value |
|-----|-------|
| `OPENAI_API_KEY` | `sk-proj-eNpjyzG3Tfn...` (실제 키 전체) |
| `DATA_DIR` | `./processed_data` |
| `DAILY_COST_LIMIT` | `5.00` |

### 1-6. 배포
1. 맨 아래 **"Create Web Service"** 클릭
2. 빌드가 시작됨 (3~5분 소요)
3. 완료되면 URL이 나옴: `https://xray-agent.onrender.com`
4. 이 URL을 복사해둘 것!

### 1-7. 배포 확인
브라우저에서 접속해봐:
- `https://xray-agent.onrender.com/health`
- `{"status":"ok","uptime":...}` 나오면 성공!

---

## Step 2: Streamlit Cloud에 대시보드 배포

### 2-1. Streamlit Cloud 가입
1. https://share.streamlit.io 접속
2. **"Continue to sign in"** → **"Continue with GitHub"**
3. snowF3 GitHub 계정으로 로그인

### 2-2. New App 생성
1. 우측 상단 **"New app"** 클릭
2. 설정:

| 항목 | 값 |
|------|-----|
| **Repository** | `snowF3/streamlit` |
| **Branch** | `prod` ← 중요! |
| **Main file path** | `app/main.py` |
| **App URL** | 원하는 이름 (예: `xray-dashboard`) |

### 2-3. Advanced Settings (중요!)
**"Advanced settings"** 펼치기:

**Secrets** 에 아래 내용 입력 (TOML 형식):
```toml
AGENT_URL = "https://xray-agent.onrender.com"
```
(↑ Step 1-6에서 받은 Render URL)

**Python version**: 3.11

### 2-4. 배포
1. **"Deploy!"** 클릭
2. 빌드 시작 (2~3분)
3. 완료되면 URL: `https://xray-dashboard.streamlit.app`

---

## Step 3: 배포 확인

1. `https://xray-dashboard.streamlit.app` 접속
2. 동네 지도, 프로파일 등 탭 확인
3. 우측 하단 채팅 버튼 클릭 → AI 에이전트 테스트

## CI/CD 워크플로우

배포 후부터는 이 흐름으로 운영:

```
[개발]
  1. feat/xxx 브랜치에서 개발
  2. main에 머지
  3. dev에 머지 (테스트)

[배포]
  4. dev → prod 머지
  5. git push origin prod
  6. ✅ Render: 자동 재배포 (agent)
  7. ✅ Streamlit Cloud: 자동 재배포 (dashboard)
```

### 배포 명령어 (한 줄)
```bash
# Agent 배포
cd agent && git checkout prod && git merge dev && git push origin prod && git checkout main

# Streamlit 배포
cd streamlit && git checkout prod && git merge dev && git push origin prod && git checkout main
```

## 주의사항

### Render 무료 티어 제약
- **15분 미사용 시 서버 슬립** → 첫 요청 시 30~50초 걸림
- 월 750시간 무료 (충분)
- 512MB RAM

### Streamlit Cloud 무료 티어
- 앱 1개 무료
- Private 레포는 유료 (Public으로 변경 필요할 수 있음)
- 1GB RAM

### OPENAI_API_KEY 보안
- Render: 환경변수로 안전하게 저장됨
- Streamlit: Secrets으로 안전하게 저장됨
- ❌ 절대 코드에 하드코딩하지 말 것
